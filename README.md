# Sistema Bancário simples feito com C++

## Propósito

Este repositório é um exemplo simples de um sistema bancário brasileiro, pode ser utilizado de inspiração para outros projetos ou uma forma de ver na prática como implemetar conceitos em um projeto "real".

## Conceitos principais utilizados

- Programação orientada a objetos
- Ponteiros
- Moduralização

## Instalação

Caso queira testar o programa você vai precisar instalar ele aqui esta o passo a passo de como instalar e testar o programa:

### Requerimentos

- g++
- ninja
- cmake
- git

### Clonagem

```bash
git clone https://github.com/Dev-Benjamim/sistema-bancario.git
```

### Build

No diretório do repositório execute:

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

### Execução

No diretório da build execute:
```bash
./sistema-bancario
```
